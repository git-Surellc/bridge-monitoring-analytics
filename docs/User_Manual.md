# 结构监测数据分析系统 (v1.3.1) 用户使用手册

## 1. 系统简介
结构监测数据分析系统（Structure Monitoring Analytics System）是一套专为土木工程（桥梁、隧道、边坡等）监测数据处理而设计的智能化分析平台。它能够帮助工程师从繁杂的传感器数据中快速提取关键信息，通过算法分析和 AI 辅助，自动生成专业的技术评估报告。

**主要功能：**
*   📊 **多源数据接入**：支持 Excel 文件直接上传和 API 接口自动同步。
*   📈 **可视化分析**：提供交互式时程曲线、动态图表和关键指标高亮。
*   🧠 **智能诊断**：集成多种专业算法（如 FFT、线性回归、3σ异常识别）及 AI 大模型深度分析。
*   📝 **报告自动化**：一键生成标准 Word 格式的监测评估报告，包含图表、结论和建议。

## 2. 登录与访问

### 2.1 系统登录
系统部署在企业内网或云服务器上，请通过浏览器访问指定的 IP 地址和端口（默认为 `http://localhost:8888` 或服务器 IP）。

> **[此处插入登录界面截图]**
> *图 2-1：系统登录界面*

1.  在浏览器地址栏输入系统网址。
2.  输入管理员账号和密码（默认账号通常为 `admin`，密码请咨询系统管理员）。
3.  点击“登录”按钮进入系统主页。

## 4. 数据接入 (Data Ingestion)

### 4.1 Excel 文件上传
适用于离线数据处理场景。

1.  点击顶部导航栏的 **“数据接入”**。
2.  在左侧的“文件上传”区域，点击或拖拽 Excel 文件（支持 `.xlsx`, `.xls`）到上传框。
3.  系统会自动解析文件内容。
    *   *注意*：Excel 文件需符合特定模板格式（通常包含时间列、传感器数据列等），详细模板可参考页面下方的“Excel 模板说明”。
4.  解析成功后，点击 **“加载结果并分析”** 进入分析页面。

### 4.2 API 数据同步
适用于连接第三方监测平台（如物联网平台）获取实时数据。

1.  在“数据接入”页面右侧的“API 数据同步”区域。
2.  填写平台 API 地址、用户名和密码（如果需要）。
3.  选择要同步的 **结构物**（支持多选）和 **时间范围**（月份）。
4.  点击 **“开始同步”**。系统会在后台下载数据，任务完成后点击右侧的下载图标即可加载数据。

## 5. 分析仪表盘 (Analysis Dashboard)

这是您日常工作最频繁使用的模块。

### 5.1 结构概览与导航
页面左侧提供了 **“结构快速导航”** 面板。当您处理包含多个结构（如多座桥梁或多个隧道断面）的项目时，可以通过点击导航栏快速定位到特定结构，系统会自动滚动并展开该结构的详情卡片。

> **[此处插入左侧快速导航栏截图]**
> *图 5-1：结构快速导航*

### 5.2 图表交互与分析
每个结构的卡片中包含多个传感器的时程曲线图。

*   **查看数值**：鼠标悬停在曲线上，可查看具体时间点的数据值。
*   **缩放图表**：使用鼠标滚轮或拖拽框选，可以放大查看局部数据细节。
*   **图例开关**：点击图例（Legend）可以隐藏或显示特定传感器的数据。

### 5.3 算法工具栏
在页面顶部（或结构卡片上方），您会看到一排分析工具按钮。

> **[此处插入分析工具栏截图]**
> *图 5-2：分析算法工具栏*

*   **3σ异常识别**：自动标记超出 3 倍标准差的异常数据点。
*   **线性趋势**：拟合数据趋势线，判断监测指标是否呈现持续上升或下降趋势。
*   **FFT 频谱**：对振动数据进行快速傅里叶变换，分析频率特性。
*   **AI 智能诊断**：调用大模型对当前数据进行综合评估。

### 5.4 AI 智能总结（新功能）
在报告的 **“评估结论及建议”** 章节（通常位于页面底部），新增了 **“AI 智能生成总结”** 按钮。
点击后，AI 会自动汇总所有结构的监测情况（如传感器总数、异常数量、风险等级），生成一段精炼的项目级评估结论，并自动填充到文本框中。

## 6. 报告生成

### 6.1 报告预览与编辑
分析仪表盘的下方是报告预览区。系统根据预设模板自动生成了报告结构。
*   **编辑文本**：您可以直接修改各个章节的文本内容。
*   **插入截图**：虽然报告会自动包含图表，但您也可以手动补充说明。

### 6.2 导出 Word
1.  确认所有分析内容和结论无误。
2.  点击页面右下角的 **“生成 Word 报告”** 悬浮按钮。
3.  系统将生成 `.docx` 文件并自动下载。报告包含自动生成的目录、图表、统计表格和分析结论。

## 7. 系统配置 (AI Config)

为了使用 AI 分析功能，您需要配置 AI 模型接口。

1.  点击右上角的 **“AI 配置”** 按钮（机器人图标）。
2.  **Base URL**：输入模型服务商的 API 地址（例如阿里云百炼的 `https://dashscope.aliyuncs.com/compatible-mode/v1`）。
3.  **API Key**：输入您的 API 密钥。
4.  **模型名称**：
    *   系统支持自定义模型名称。
    *   阿里云用户推荐使用 `qwen-turbo`, `qwen-plus` 或 `qwen-max`。
    *   Coding 平台用户请使用 `qwen-2.5-coder-32b-instruct` 等专用模型。
    *   *提示*：如果不知道模型名称，可以尝试点击输入框右侧的“获取可用模型”按钮（仅限支持该接口的服务商）。
5.  点击 **“保存配置”**。

## 8. 常见问题 (FAQ)

**Q: AI 分析提示 "Model not supported" 怎么办？**
A: 请进入 AI 配置页，手动修改“模型名称”。例如阿里云百炼不支持 `gpt-3.5-turbo`，请改为 `qwen-turbo`。

**Q: 为什么生成的报告里图片显示不全？**
A: 请确保在点击生成报告前，页面上的图表已经加载完毕。系统是基于当前页面渲染状态生成截图的。

**Q: 如何批量删除旧文件？**
A: 进入“数据管理”页面，勾选需要删除的文件（支持按 Shift 键多选），然后点击右上角的“批量删除”按钮。
